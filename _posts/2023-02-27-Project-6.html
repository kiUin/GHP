Project 6: Personalized Wake-Up Alarm That Forces to Wake You Up

I built this alarm clock as a final project for the ELEGOO super starter kit that I bought.

Alarm clock features:
- NO RTC MODULE (this is the challenge)
- Very customizable
- Shows current time (using millis() )
- IR remote controlled
- Includes LED lights
- Active buzzer that plays 'Despacito' (for jokes)

What I learned:
- Programming in C++ (Arduino IDE)
- Schematics
- Circuitry and Circuit Design
